# RabbitGather

# 原始構想

```
社群網路軟體

1. 以地區做自動篩選 (意即在此地才能接受到此地發布之相關文章)
2. 可自行選擇收聽範圍，概念類似電台

優點 : 社區化、資訊更鄰近於生活圈，並非一打開皆是全部文章
賣點 : 可標記兩三個點為中心，擴張多少範圍，標記點間可分類篩選也可合併 (ex 公司往外3公里，住家往外1公里)
```
